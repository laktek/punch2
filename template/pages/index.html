Hello, welcome to Punch

This is your fresh new Punch site. It is already setup with essentials needed for a personal blog.

Let's take 5 minutes to customize it:

  • Edit this page, by opening pages/index.html.
  • You can define reusable sections in pages as partials. To edit the footer, open partials/footer.html.
  • Punch is already setup with Tailwind for easily styling the pages. To customize the styles, edit css/main.css.
  • To change the site title, open contents/site.json.
  • Time to write your first blog post - Open a sample post in contents/blog and edit it.

Check Punch docs for a detailed getting started guide.
